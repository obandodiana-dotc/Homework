About this Website

My Travel Adventures is a single-page website created as a web development exercise. The goal is to practice building a complete UI: navigation, hero section, content blocks, image gallery and a contact area.

It uses HTML5 + Tailwind CSS + a bit of JavaScript to create a modern, responsive, and interactive interface without heavy frameworks.

The content is based on imagined travel moments, but the layout and structure reflect patterns you can reuse later for portfolios, landing pages, and personal projects.

What this UI includes

  • Sticky navigation bar with smooth scrolling.
  • Hero section with call-to-action button.
  • Destination cards with hover animations.
  • Responsive image gallery with overlays.
  • Simple contact form block.
  • Dark mode support (toggle at the top).
