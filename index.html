About this project

My Travel Adventures was built as a web development exercise. It focuses on:

  • Creating a multi-page layout with a shared navigation bar.
  • Designing hero sections, cards and image galleries.
  • Using Tailwind CSS for quick, responsive styling.
  • Practicing UI thinking with a simple, fun theme (travel).
